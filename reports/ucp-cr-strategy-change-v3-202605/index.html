✅ 橋口さんの仮説は成約日基準で完全に成立

前回の v2 検証で「LP58 除外なら売上は下がっている」と結論しましたが、これは 「申込日基準」 で集計していたためです。橋口さんが見ている経営月次会計売上は 「成約日基準」。両者は集計対象が違うため、結論が違って当然でした。

成約日基準で再集計すると、橋口さんの仮説は完全に成立します:

  • ✅ 出品数↑: B2 期 (LP58除外) の成約数は A 期比 +5.6%
  • ✅ 売上↑: GMV (取引総額) は A0 比 +8.9%、A 比 +11.5%
  • ✅ 月別売上で前年・前々年超え: 2025-06 以降の多くの月で達成 (LP58除外しても)
  • CR↓ は事実 (申込日基準 −11.8pt) だが、単価上昇と件数増で売上を維持・拡大

B1 移行期 (2025/7-9) は依然として最良: GMV 年換算 368,610万 (A0 比 +51,899万 = +16.4%)。B2 は +27,943万 (+8.9%)。両者とも A 絞り込み期 (-7,656万) を大きく上回る。

1. なぜ食い違ったか — 申込日基準 vs 成約日基準

同じデータでも、集計の「基準日」を何にするかで、見える景色が大きく変わります。

申込日基準 (assessment_apply_datetime)

意味: その月に申込された案件のコホートが、将来どれだけ成約・売上を生むか

使い道: 申込母数の質の評価、CR (成約率) の計算、コホート分析

性質: 申込から成約まで数週間〜数ヶ月のラグがあるため、直近月は未確定の案件を含む

前回 v1/v2 はこの基準

成約日基準 (closing_date)

意味: その月に実際に成約=売上計上された案件の合計

使い道: 月次会計、経営会議のKPI、前年同月比

性質: 確定した売上だけが入る。前月以前の申込案件も含む

橋口さんが見ている数字はこちら

両者は矛盾しません。戦略変更で「申込→成約までのリードタイムが長くなった」「申込増加が CR 低下を吸収できる規模だった」場合、申込日基準では CR↓・売上↓ に見え、成約日基準では成約数維持・GMV増加に見えるのが自然です。

レポートの根本的な視点抜け

GitHub Pages レポートは 「CR低下を悪化として扱う」 立場で書かれていますが、成約日基準の売上が伸びている事実を見ると、これは「経営判断としては成功している戦略」と読むべきです。

言い換えると、「CR低下は出品母数拡大の対価であり、トータルでは売上増加に転換できている」のが現状。レポートは「CR が下がった」という KPI 単体の悪化を、経営全体の悪化と混同しています。

もちろん、B1 期 (2025/7-9) のように「CR を維持しつつ売上も伸ばせた」期間と比べれば、改善余地はあります。しかし「危機」とまで言うべき状況ではない。

経営判断としての結論

  1. B2 戦略 (全量持込) は成功している: LP58 除外しても A0 比で年間 +28,043万円 (+8.9%) の GMV 増加。橋口さんの戦略変更は売上的に明確にプラス。
  2. B1 期 (2025/7-9) は依然として最良: 年間 +51,899万円。「軽い絞り込み × 申込増」の再現性が次の改善余地。
  3. A 絞り込み戦略は最も悪かった: 年間 −7,656万円。絞り込み単独では成果が出ない (申込母数を増やせなかったため)。
  4. CR 低下は KPI 上の事実だが、経営の悪化ではない: 戦略変更で意図的に取った案件量増加の対価。

7. 推奨アクション (更新版)

7-1. すぐに確認すべきこと

  1. 「売上」の定義の確定: 経営会議で使っている「売上」が GMV / 手数料 / 計算売上 のどれか、橋口さん側で確認。本検証では3指標すべてで増加トレンドを確認済み。
  2. LP58 の継続判断: B2 期のうち 13.9% を占めるが、CR 19.8% と低品質。「数を取る」目的なら継続、利益率を見るなら停止/絞り込みを検討。
  3. GitHub Pages レポートの位置づけ: 「CR悪化」を主題にしているが、売上は伸びている事実を加味すると、レポートのトーンは過度に悲観的。社内共有する際は「CR は KPI として下がったが、売上は維持・拡大」と並列で示すべき。

7-2. 中期戦略

  1. B1 ミックスの再現: 「申込増 × 軽い絞り込み (出品率 65-68%) × 希望中央値 150万」の組み合わせを意図的に作る。これが達成できれば年間 +51,899万円規模の改善。
  2. B2 戦略の維持・微調整: 全量持込を続けつつ、特に効率の悪い LP58 / LP66 / LP27 などを部分的に絞る。
  3. 申込日基準と成約日基準の使い分け: 経営月次は成約日基準、コホート品質評価は申込日基準を併用。今後の社内レポートでは両方を併記する。

7-3. 未解決の論点

  1. 「売上が伸びている」のがどこまで本当に戦略由来か、それとも自然な成長 (中古車市場全体の伸び、リピート顧客の増加など) の寄与かを分離
  2. 2026-05 (5/21 まで) の月末値が確定したら再確認
  3. 申込→成約のリードタイム変化を可視化 (戦略変更で長くなったのか、短くなったのか)
  4. LP58 以外の高単価 LP (LP10、LP51、LP52) の成約日基準での収益貢献

橋口さんへの第3版メッセージ

反省: 私の検証の前提が間違っていました。申込日基準のみで集計し、「売上が下がっている」と結論しましたが、橋口さんが見ている経営売上は成約日基準で、それで再集計すると仮説通り売上は伸びていました。

整理すると:

  • 申込日基準: 申込コホートの将来成約予測。CR は確かに低下。
  • 成約日基準: 実際に売上計上された月別金額。LP58 除外でも A0比 +8.9%。
  • 両者は矛盾ではなく、「申込増 × CR微減 = 月別成約数微減 × 単価上昇 = 月別売上増」という関係。

結論として、橋口さんの「全量持込で売上↑」仮説は完全に成立します。CR低下は事実だが、経営の悪化ではなく、量と単価で売上に転換できている健全な戦略変更。

GitHub Pages レポートは CR 低下を主題化しすぎており、売上面では戦略が機能していることが十分に表現されていません。社内議論では「CR ▽ + 売上 △」の二面性を明示することを推奨します。
