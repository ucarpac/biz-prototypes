OneSignal KPI Dashboard

OneSignal の登録数、通知許諾、配信到達、クリックまでを同じ面で追えるようにしたレポートです。 現時点では 2026-05-19 時点のスナップショットを中心に整理し、履歴系列は GitHub Actions からの自動更新で今後拡充する前提にしています。

Core Metrics

DL母数から OneSignal 登録、通知許諾、到達、クリックまでのファネルを一度に見られるようにしています。

Trend View

現在は確定値のみを優先して反映しています。履歴データはワークフローによる追記を前提にした構成です。

Counts

DL母数、OneSignal登録、通知許諾の推移

Rates

登録率、許諾率、到達率、CTR を同じ尺度で比較

Notes & Source

この集計の前提、未確定項目、GitHub へ載せたソースをまとめています。

前提と未確定事項

  • 有効DL累計は OneSignal テーブルだけでは確定できないため、現時点では `33,855` をフォールバック値として利用しています。
  • Push経由DL数は今回のスナップショットに未反映です。AppsFlyer や AppDB と結合する拡張余地があります。
  • クリック数は `onesignal_notifications_raw.raw_json.clicked` を利用しています。
  • GitHub Actions は `push_notification_dashboard_data.js` を更新し、表示側の HTML は固定のまま使い回せる構成です。
